    • JavaScript面向对象(一)

    面向对象程序设计是目前大规模程序设计中的基本程序设计方法，通过识别系统中的数据和与数据有关的操作，将系统抽象成一个一个的对象与他们之间的协作，能使系统降低耦合性，增加内聚性。

    不同于 C++ / Java 等主流面向对象编程语言，JavaScript 语言中的面向对象使用了一种基于原型的方式来实现。在JavaScript 语言中，不存在 “类” 这个概念（尽管在 ES6 标准中引入了 class 关键字，但是显然 JavaScript 中的 class 与 C++ / Java 等语言中的 class 相去甚远），而可以说是真正的实现了 “万物皆对象”。

    JavaScript 使用了作用域链和原型链相结合的方式实现了传统面向对象语言中 “类” 支持的若干概念，接下来就详细地看一看。

    在这里有一个特别神奇的规则：

    1. 使用new操作符调用函数，如果该函数没有return语句，则使用新对象作为函数返回值
    2. 如果该函数有return语句：
      1. return语句返回的是简单类型对象，则忽略此return语句，使用新对象作为函数返回值
      2. return语句返回的是复杂类型对象，则以此对象作为函数返回值
